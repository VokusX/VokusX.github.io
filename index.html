MOBILE DEVELOPMENT

SwiftWeather

Initally, the project was intended for educational purposes to teach myself Swift and how to use XCode to build user-focused, responsive apps, but grew into much more. The app uses many APIs and libraries to fetch data, such as Apple's Corekit for user location, as well as OpenWeatherMap for fetching weather data using Alamofire, and parsing it. The end result is a material design app that is simple, yet beautiful.

Source available on GitHub

To the Stars

To the Stars is a 2D side-scroller inspired by fast-paced and challenging games, such as Geometrey Dash. The game was made using Unity, with all of the coding done in C# and Python, with an emphasis on optimized code for running on a variety of devices. The game takes advantage of procedurally generated content to provide users with an endless, always new and exciting experience.

Source available on GitHub
